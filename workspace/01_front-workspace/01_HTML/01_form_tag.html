form 태그

form 태그 내에 submit 클릭시
해당 form 태그 내에 작성된 사용자가 입력한 값들을 서버로 전달하면서
요청하는 역할 수행



*action : 요청할 서버 주소(경로)
=> 해당 서버로 사용자가 form 내에 입력 태그들을 통해 작성한 값들을 전달
*method : 요청 전송 방식 지정(get/post) 기본값(defult): get
=> get 방식 : 요청 시 사용자가 입력한 값들이 url에 노출되는 방식
=> post 방식 : 요청 시 사용자가 입력한 값들이 url에 노출되지 않는 방식

사용자가 입력한 값들은 key=value 형태로 전달하면서

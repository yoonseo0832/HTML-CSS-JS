라디오 버튼과 체크박스 관련 input 태그의 타입

라디오 버튼

  • type="radio"
  • 제시한 값들 중 하나만 선택할 수 있도록 할때 사용됨
    • >name 속성 값을 같도록 설정해야 함 => 요소끼리 하나의 그룹으로 묶어진다
  • form 전송 시(submit) 현재 선택된 라디오 버튼의 value 속성값이 전달됨
    • 즉, 해당 라디오 버튼 선택 시 전달할 값을 value 속성에 반드시 명시
성별:

체크박스

  • type="checkbox"
  • 제시한 여러 개의 값들 중 여러 개를 선택할 수 있도록 하고자 할때
  • 선택된 체크박스 value 속성 값 전달 => 반드시 value와 name 속성값을 지정해야함

select, option 태그

  • 사용자가 직접 값을 입력하는 것이 아니라 제시한 여러 값들 중 선택할 경우
  • option와 select 태그 모두 시작과 끝 태그 있음
  • 제출 시 선택된 option 태그의 값 전달
  • 각 option 태그의 value 속성이 지정되지 않는 경우 content 영역의 값이 전달됨
  • 각 option 태그의 value 속성이 지정되어 있으면 해당 value 속성 값이 전달됨
언어:

datalist 태그

  • 사용자에게 직접 값읋 입력받거나 select처럼 목록을 통해 선택 받는 태그

변수와 자료형

변수 선언 시 let, const, var라는 키워드를 이용해 선언 가능

* 변수 선언 방법

  • let : 중복 선언 불가능, 재할당 가능
  • const : 중복 선언 불가능, 재할당 불가능
  • var : 중복 선언 가능, 재할당 가능

*자료형

변수 선언 시 자료형을 변도로 지정하지 않음 => ( 자료형의 개념이 없는 것은 아님 )
=> 인터프리터(한줄 씩 실행되는) 언어 특성상
변수에 대입되는 값에 따라서 알아서 자료형 결정

자바스크립트의 자료형

  • 숫자 : Number
  • 문자열 : String
  • 논리값 : Boolean
  • 초기화되지 않은 것 : undefined
  • 값이 없는 것: null
  • 객체/배열/함수 : Object

*변수 명명 규칙

  1. 일반적으로 소/대문자 사용, 문자/숫자 모두 사용 가능
  2. 특수 문자는 언더바(_)와 달러기호($)만 사용 가능
  3. 숫자로 시작할 수 없음
  4. 키워드(예약어)를 변수명으로 사용할 수 없음 EX) let, const, var...

표기법 (naming type)

  1. camelCase: 낙타 표기법, 자주 사용되는 표기법 소문자로 시작하고 다른 단어가 시작할때마다 대문자로 구분하는 방법 ec) user id ==> userId
  2. snake_case: 파이썬에서 주로 사용 표기법 단어마다 _(언더바)로 구분 ex) user id ==> user_id
  3. PascalCase : c# 또는 ms 계열의 언어에서 사용되는 표긱법 모든 단어의 시작을 대문자로 구분한다 ex) user id ==> UserId

목록 관련 스타일(list~~)

list-style-type : 블릿 기호 변경하고자 할때 사용하는 스타일

list-sytle-image : 블릿 기호 이미지를 적용하고자 할 때 사용

list-style-position : 블릿 기호의 위치 조정할 때 사용

        *순서 없는 목록(ul)
        선택자{
            list-style-type : disc | circle | square | none;
            list-style-image : url("적용하고자하는 이미지 경로");
            list-style-position : inside | outside;
        }
        *순서 있는 목록(ol)
        선택자{
            (숫자)list-style-type : decimal | decimal-leading-zero;
            (알파벳)list-style-type : lower-alpha | upper-alpha;
            (아라비아숫자/로마자)list-style-type : lower-roman | upper-roman;
        }

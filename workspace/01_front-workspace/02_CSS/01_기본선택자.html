기본 선택자

선택자란?

특정 html 요소를 선택하려고 사용, 해당 요소 선택해 원하는 스타일과 기능을 적용 가능

모든(전체) 선택자 : *

현재 문서 내 모든 요소 선택할 때 사용함

        *{
            스타일속성: 값;
            스타일속성: 값;
        }

태그 선택자 : 태그명

현재 문서 내 해당 태그를 모두 선택하고자 할 때 사용

        태그명{
            스타일속성: 값;
        }

아이디 선택자 : #아이디명

현재 문서 내에 특정 html 요소 하나만을 선택하고자 할 때 사용
해당 요소에서 id속성을 이용하여 고유한 값을 부여해야함 

        #아이디명{
            스타일속성:값;
        }
  1. 데이터a
  2. 데이터b
  3. 데이터c
  4. 데이터d

클래스 선택자 : .클래스명

현재 문서 내 특정 html 요소 "여러 개"를 선택하고자 할 경우

        .클래스명{
            스타일속성:값;
        }
